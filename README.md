# News

## Feb. 22, 2023 

参考文献の表示の仕方を変更しました。

途中から変えたので表記がごちゃごちゃになっている可能性があります。

旧版は
```
サイト名 「ページタイトル」 著者名
```
でしたが、

新バージョンは
```
著者名. "ページタイトル". サイト名.
URL, (cited: 閲覧日)
```
というように、英文の引用の形式に倣いました。

よろしくお願いします。
